ЗАПИСАТЬСЯ

О СПЕЦИАЛИСТЕ

Мой подход — сочетание медицинской точности и эстетического вкуса:

1. Здоровье, как основа любой эстетической коррекции

2. Красота, подчёркивающая естественность и уникальность каждого пациента

3. Результат, основанный на доказательной медицине и индивидуальном подходе

Мой принцип:

"Здоровая кожа — основа красоты. Сочетаю лечение дерматологических заболеваний с эстетической коррекцией для естественного и гармоничного результата".

ЗДОРОВЬЕ КОЖИ

Дерматология

Терапия любой степени тяжести, работа с акнеформными дерматозами. Индивидуальный подбор схемы лечения с учётом особенностей кожи.

Комплексный подход к лечению сложных хронических состояний кожи. Подбор терапии и ухода для длительной ремиссии.

Профессиональный осмотр невусов и других новообразований кожи. Ранняя диагностика для вашего спокойствия.

Безопасное удаление доброкачественных новообразований кожи современными методами.

ЭСТЕТИКА И ОМОЛОЖЕНИЕ

Косметология

Гармонизация лица: скулы, подбородок, губы. Работа с препаратами гиалуроновой кислоты для естественного и выразительного результата.

Работа с качеством кожи с помощью полимолочной кислоты. Стимуляция собственного коллагена для естественного омоложения.

Коррекция мимических морщин: лоб, межбровье, область вокруг глаз. Естественный результат без эффекта маски.

Микроигольчатый RF-лифтинг на аппарате Vivace — золотой стандарт лифтинга и сужения пор. Подтяжка кожи без хирургии.
